Map Selection Panel Demo

Map Selection Panel Features

  • 6 Unique Maps: Classic Maze, Speed Circuit, Serpentine Path, Grid Lock, Spiral Madness, and Island Hopper
  • Visual Previews: Each map shows a mini preview with difficulty rating
  • Map Rotation: Automatic rotation every 3 races with counter display
  • Interactive Selection: Click to select, hover for preview
  • Random Selection: Let the system pick a map for you
  • Keyboard Support: Full keyboard navigation and shortcuts
  • Responsive Design: Works on desktop, tablet, and mobile

Keyboard Shortcuts:

M Toggle Map Selection | Enter Confirm | Esc Cancel | R Random Map
